# 🗺️ Athos Maps - Offline Πλοήγηση στο Άγιο Όρος

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Made with Maplibre](https://img.shields.io/badge/Made%20with-Maplibre-00BCD4.svg)](https://maplibre.org/)

> Πλήρως offline εφαρμογή πλοήγησης για το Άγιο Όρος με 140+ διαδρομές.

## ✨ Χαρακτηριστικά

- 🗺️ **OpenFreeMap** - Δωρεάν χάρτες
- 📍 **GPS Tracking** - Παρακολούθηση θέσης
- 🧭 **Follow Mode** - Αυτόματη παρακολούθηση
- 🛤️ **140+ Διαδρομές** - Όλες οι διαδρομές του Αγίου Όρους
- ⭐ **Αγαπημένες** - Online & Offline αποθήκευση
- 🆘 **SOS** - Έκτακτη ανάγκη
